# Online Quiz Application

## Project Overview

The Online Quiz Application is a web-based application developed using Java, JSP, Servlets, MySQL, and Apache Tomcat. It allows users to log in, attempt quiz questions, submit answers, and view their final score.

This project was developed to understand web application development concepts, database connectivity, user interaction, and server-side processing.

---

## Features

- User Login System
- Multiple Choice Quiz Questions
- Score Calculation
- Result Display
- Database Connectivity using MySQL
- Interactive User Interface

## How It Works

1. User opens the application.
2. User logs into the system.
3. Quiz questions are displayed.
4. User submits answers.
5. The application evaluates the answers.
6. Final score is displayed.
